# CPSC5860_final_project

[pics](./pics/) are result pictures generated from this project.

[code](./code/):
1. [graph_nn.py](./code/graph_nn.py) is used for graph structure built, train, test and evaluate.
2. [mlp.py](./code/mlp.py) is used for basline MLP model.
3. [plot.py](./code/plot.py) is used for plot generating.
4. [simu.py](./code/simu.py) is used for simulated data since our data is not so balanced.
5. [xgb.py](./code/xgb.py) is used for xgboost baseline model. Also, [xgboost_baseline.ipynb](./code/xgboost_baseline.ipynbsx)
6. [model_train.ipynb](./code/model_train.ipynb) is the main processing file.
